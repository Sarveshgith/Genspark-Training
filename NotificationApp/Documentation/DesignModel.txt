MODEL/ DATA LAYER
    User => Model class
    Notification => Model class

INTERFACE LAYER
    Notification => Interface  
        Send(User, Notification)

    Notification Interface implemented by => Email, SMS Classes

    IRepository => Defines CRUD Operations 

REPOSITORY LAYER
    Repository => Implements the IRepository Interface but Defines Create as Abstract 
            as it can be used by both Notification and User for Creating objects

    //Implemented only for Demo Purpose. Optional
    UserRepository => Implements Repository where it Overrides Create Method for Custom User Creation

    NotificationRepository => Implements Repository where it Overrides Create Method for Custom Notification Creation

SERVICE LAYER
    Notification Service =>
        User input
        Validations
        Send Data to Repository

CLIENT 
    Program.cs => Client-side Interface 
            Menu Display and Service Layer accessing
