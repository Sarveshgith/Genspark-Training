### 🔑 Authorization Policy Mappings

#### 1. **Auth Endpoint (`api/auth`)**
* `POST /register`: `[AllowAnonymous]` *(Public signup)*
* `POST /login`: `[AllowAnonymous]` *(Public authentication)*

---

#### 2. **User Endpoints (`api/users`)** — Class-level `[Authorize(Policy = "AdminOnly")]`
* `GET /`: `[Authorize(Policy = "AdminOnly")]` *(Requires **Admin** role)*
* `GET /{id}`: `[Authorize(Policy = "AdminOnly")]` *(Requires **Admin** role)*
* `GET /roles`: `[Authorize(Policy = "AdminOnly")]` *(Requires **Admin** role)*
* `PUT /{id}`: `[Authorize(Policy = "AdminOnly")]` *(Requires **Admin** role)*
* `PATCH /{id}/password`: `[Authorize(Policy = "AdminOnly")]` *(Requires **Admin** role)*
* `DELETE /{id}`: `[Authorize(Policy = "AdminOnly")]` *(Requires **Admin** role)*

---

#### 3. **Table Endpoints (`api/tables`)** — Class-level `[Authorize]`
* `GET /`: `[Authorize]` *(Requires any valid token)*
* `GET /{id}`: `[Authorize]` *(Requires any valid token)*
* `POST /`: `[Authorize(Policy = "AdminOnly")]` *(Requires **Admin** role)*
* `PUT /{id}`: `[Authorize(Policy = "AdminOnly")]` *(Requires **Admin** role)*
* `PATCH /{id}/status`: `[Authorize(Policy = "AdminOnly")]` *(Requires **Admin** role)*
* `DELETE /{id}`: `[Authorize(Policy = "AdminOnly")]` *(Requires **Admin** role)*

---

#### 4. **Category Endpoints (`api/categories`)** — Class-level `[Authorize]`
* `GET /`: `[AllowAnonymous]` *(Public menu reads)*
* `GET /{id}`: `[AllowAnonymous]` *(Public menu reads)*
* `POST /`: `[Authorize(Policy = "AdminOnly")]` *(Requires **Admin** role)*
* `PUT /{id}`: `[Authorize(Policy = "AdminOnly")]` *(Requires **Admin** role)*
* `DELETE /{id}`: `[Authorize(Policy = "AdminOnly")]` *(Requires **Admin** role)*

---

#### 5. **Menu Endpoints (`api/menu`)** — Class-level `[Authorize]`
* `GET /`: `[AllowAnonymous]` *(Public menu reads)*
* `GET /{id}`: `[AllowAnonymous]` *(Public menu reads)*
* `POST /`: `[Authorize(Policy = "AdminOnly")]` *(Requires **Admin** role)*
* `PUT /{id}`: `[Authorize(Policy = "AdminOnly")]` *(Requires **Admin** role)*
* `PATCH /{id}/availability`: `[Authorize(Policy = "AdminOrChef")]` *(Requires **Admin** or **Chef** role)*
* `DELETE /{id}`: `[Authorize(Policy = "AdminOnly")]` *(Requires **Admin** role)*

---

#### 6. **Order Endpoints (`api/orders`)**
* `GET /`: `[Authorize(Policy = "AdminOnly")]` *(Requires **Admin** role)*
* `GET /active`: `[Authorize(Policy = "AdminOrChef")]` *(Requires **Admin** or **Chef** role)*
* `GET /{id}`: `[Authorize(Policy = "AdminOnly")]` *(Requires **Admin** role)*
* `POST /`: `[Authorize(Policy = "CanPlaceOrder")]` *(Requires **Guest Session**, **Admin**, or **Customer**)*
* `POST /{orderId}/items`: `[Authorize(Policy = "CanPlaceOrder")]` *(Requires **Guest Session**, **Admin**, or **Customer**)*
* `DELETE /{orderId}/items/{itemId}`: `[Authorize(Policy = "CanPlaceOrder")]` *(Requires **Guest Session**, **Admin**, or **Customer**)*
* `PATCH /{orderId}/status`: `[Authorize(Policy = "AdminOrChef")]` *(Requires **Admin** or **Chef** role)*
* `PATCH /{orderId}/assign`: `[Authorize(Policy = "AdminOnly")]` *(Requires **Admin** role)*

---

#### 7. **Bill Endpoints (`api/bills`)** — Class-level `[Authorize]`
* `POST /`: `[Authorize(Policy = "AdminOnly")]` *(Requires **Admin** role)*
* `GET /`: `[Authorize(Policy = "AdminOnly")]` *(Requires **Admin** role)*
* `GET /order/{orderId}`: `[Authorize(Policy = "CanPlaceOrder")]` *(Requires **Guest Session**, **Admin**, or **Customer**)*
* `PUT /{id}`: `[Authorize(Policy = "AdminOnly")]` *(Requires **Admin** role)*
* `PATCH /{id}/status`: `[Authorize(Policy = "AdminOnly")]` *(Requires **Admin** role)*
* `GET /order/{orderid}/pdf`: `[Authorize(Policy = "CanPlaceOrder")]` *(Requires **Guest Session**, **Admin**, or **Customer**)*
